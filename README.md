📝To-Do List App
A sleek and modern to-do list app built with React and styled using Tailwind CSS. Features include dark mode, task management, and persistent storage.

🚀 Features
Add, Edit, and Delete Tasks
Mark Tasks as Complete
Dark Mode Toggle 🌗
Persistent Storage (Local Storage)
Responsive Design

🛠️ Tech Stack
React
Tailwind CSS
Font Awesome Icons
Google Fonts
